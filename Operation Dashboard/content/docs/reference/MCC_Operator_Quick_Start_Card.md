# MCC VR CCTV Training System

Operator Quick Start

## Hardware Setup

☐ Power on your PC. Confirm Windows desktop is loaded.

☐ Confirm SteamVR and VIVE Hub show green status.

☐ Plug in the HTC Vive Focus Vision via the link cable. Confirm it's charged.

☐ Put on the VR headset NOW. You'll keep it on for the entire session.

☐ Strap the EmotiBit sensor onto your wrist. Confirm the LED is solid (not blinking).

☐ The application configuration should already be set by IT. If this is first-time setup, ask IT to verify.

## Session Workflow

1. Launch the MCC application (auto-connects to supervisor)

2. Enter your name in the lobby screen

3. Wait in the lobby (you'll see operator count and connected names)

4. Eye tracking calibration runs when the supervisor starts the scenario

5. Monitor CCTV feeds, mark suspects, respond to alarms

6. Session ends. Remove headset and EmotiBit. Data saves automatically.

7. Review your performance report with the supervisor.

### Key Reminders

* CCTV Monitor: 4-camera grid. Click to go fullscreen, click again for grid view.
* Marking: Tag suspicious individuals when you spot them. Everything is timestamped.
* Alarms: Respond as fast as possible. Your response time is scored.
* Your composite score (0-100) is built from: Detection Accuracy (40%), False Positives (20%), Timeliness (20%), Attention and Gaze (20%).

If anything doesn't work, notify the instructor or IT staff immediately.
