# MCC VR CCTV Training System

Supervisor Quick Start

## Session Hosting

☐ Power on supervisor PC (no VR headset needed).

☐ Verify server.json is configured (IT should have set this up).

☐ Launch the MCC application.

☐ Select a scenario from the dropdown (Airport, Embassy, Mall, Concert, Protest).

☐ Click "Create Scenario" to start hosting.

☐ Wait for operators to connect (check names and count in lobby).

☐ Click "Start Scenario" when all operators are ready.

## Dashboard and Reports

### Live Monitoring Panels

* Operator Feed: See what any operator sees. Switch between operators by clicking their name. Live wall shows all operators at once.
* Biometric Panel: Heart rate (bpm), EDA (skin conductance), skin temperature, and SCR stress indicator.
* Eye Tracking: Gaze overlay on camera view. Heatmap builds over time showing attention distribution.
* Distractions and Alarms: Trigger distractions or emergency alarms mid-scenario to test operator focus and response.

### After the Session

1. Open the post-session Dashboard

2. Select an operator to review

3. Review tabs: Eye Data, Biometrics, Marking Data, Activity Logs, Alarms

4. Use the Replay System (scrub timeline, adjust speed 0.5x/1x/2x, toggle overlays)

5. Click "Generate Report" to create the PDF performance report
