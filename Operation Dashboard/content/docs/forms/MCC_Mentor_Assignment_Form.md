Section 4: Mentor Responsibilities

The assigned mentor agrees to:

* Guide the new hire through all four onboarding phases as described in MCC-SOP-001
* Be available during scheduled onboarding hours for questions and observation
* Complete the Competency Observation Checklist during Phase 3 (Solo Practice)
* Provide constructive feedback after each practice session
* Recommend sign-off only when the new hire demonstrates consistent competency
* Escalate concerns to the Training Manager if progress falls behind schedule

Section 5: New Hire Responsibilities

The new hire agrees to:

* Complete all activities in each onboarding phase
* Review printed materials (User Manual, Quick Reference Card, Troubleshooting Flowchart)
* Ask questions when unsure rather than guessing
* Aim for a composite score of 70% or above on the performance report
* Participate in the sign-off assessment on Day 8
